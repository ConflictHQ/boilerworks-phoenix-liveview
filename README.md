# Boilerworks Phoenix + LiveView

> Real-time Elixir applications with Phoenix 1.7 and LiveView for collaborative and interactive experiences.

**Status:** Planned

Phoenix with LiveView for applications that demand real-time updates, collaborative editing, or persistent WebSocket connections. The BEAM VM gives you fault tolerance and massive concurrency out of the box. Choose this for chat apps, live dashboards, collaborative tools, and anything where server-pushed updates are the core UX.
